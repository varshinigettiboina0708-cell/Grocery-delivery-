# FreshCart - Grocery Delivery App UX Case Study

## Overview
FreshCart is a grocery delivery mobile app UX project focused on creating a simple and efficient grocery shopping experience.

## Problem Statement
Users face difficulties with:
- Finding products quickly
- Complicated checkout process
- Lack of order tracking

## UX Process
1. User Research
2. Persona Creation
3. Empathy Mapping
4. User Flow
5. Wireframing
6. UI Design
7. Prototype Testing

## Tools Used
- Figma
- FigJam
- Canva

## Key Features
- Product search
- Categories
- Cart management
- Secure payment
- Order tracking
